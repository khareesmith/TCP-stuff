How to make a sandwhich. WITH STEP-BY-STEP INSTRUCTIONS!

  1. Choose favorite type of bread for the sandwich.
  2. Spread condiments directly onto the bread.
  3. Add meats and cheeses on bottom of sandwich.
  4. Add vegetables to sandwich for more texture.
  5. Season sandwich for enhanced flavor.
  6. (Optional) Toast sandwich if desired hot or crunchy.
  7. Enjoy!
